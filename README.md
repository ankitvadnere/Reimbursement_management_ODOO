# Reimbursement Management App

A complete Reimbursement Management System built with React and Tailwind CSS.

## Features
- **Role-based Access**: Admin, Manager, and Employee dashboards.
- **Employee Portal**: Create and track expense reimbursements with multi-currency support (INR, USD, EUR, etc.).
- **Manager Portal**: Review and approve/reject expense requests with inline comments.
- **Admin Portal**: Manage users, roles, and configure complex approval workflows.
- **Modern UI**: Built with a clean, navy-themed professional interface.

## Tech Stack
- React
- Tailwind CSS
- Vite

## Getting Started
1. `npm install`
2. `npm run dev`
